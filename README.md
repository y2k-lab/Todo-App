# Todo App

Node.js,Express,SQLiteを使用して作成したTodo管理アプリです。

##機能

-タスク追加
-タスク編集
-タスク削除
-タスク完了　/ 未完了切り替え
-SQLiteによるデータ保存

##使用技術

-HTML
-JavaScript
-Node.js
-Express
-SQLite3
-CORS

##学習内容

このアプリでは以下を学習しました。

-CRUD処理
-REST API
-Expressによるサーバー構築
-SQLiteを使ったデータベース操作
-fetchを使った非同期通信

##画面

Todoタスクの追加、編集、削除、完了管理が可能です。
